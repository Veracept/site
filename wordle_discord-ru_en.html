Russian Wordle Discord Bot

This is my first blog post and it starts right off with a Discord bot, enjoy reading :)

Some information about the bot

This bot was written in the Python programming language using the Nextcord library. The bot works according to the principle of checking words from the so-called database, which is located in the bot settings file itself :)

If you are interested in viewing the code or somehow improving the bot, then follow the link below to the GitHub repository. There are instructions on how to set it up.

Go to GitHub

How the bot works

There are 4 commands in the bot, the functionality of which is described on the side, and if you follow the link below, you can find all the words that are used in the bot

All words

/play daily

This command allows you to play the Daily Hidden Word. This command allows you to play the Daily Hidden Word.

/play random

This command allows you to play a random hidden word.

/play id

This command allows you to select game IDs, there are 1.OOO in total.

/info
